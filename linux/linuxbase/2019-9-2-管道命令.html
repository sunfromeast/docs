# 截取命令

# cut

//cut使用格式，cut处理的是行级数据
$ cut -d '分割字符' -f fields
$ cut -c 字符范围
1
2
3

其中：

-d ,-f：一起使用

-f：选出第几段

-c： 以字符的单位选出固定区间

使用示例：

$ echo $PATH
/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/root/bin

$ echo $PATH | cut -d ':' -f2
/usr/local/bin

$ echo $PATH | cut -d ':' -f2,3
/usr/local/bin:/usr/sbin

# grep

$ grep [-acinv] [--color=auto] '查找的字符串' filename
1

其中：

-a：二进制文件以text文件方式查找数据

-c：统计找到“查找的字符串”的次数

-i：忽略大小写

-n：输出行号

-v：反向选择，显示出没有“查找的字符串”

# 排序命令

# sort

$ sort [-fbMnrtuk] [file or stdin]
1

其中：

-f：忽略大小写

-b：忽略最前面的空格

-M：以月份排序

-n：纯数字排序

-r：反向排序

-u：行唯一

-t：分割符，默认Tab分割

-k：哪个区间排序

使用示例：

//对/etc/passwd按:分割以第三列来排序
$ cat /etc/passwd | sort -t ':' -k 3

//输出的数据仅去账号
$ last | cut -d ' ' -f1 | sort
1
2
3
4
5

# uniq

重复的数据只显示一个

$ uniq [-ic]
1

​ 其中：

-i：忽略大小写

-c：计数

使用示例：

//last列出账号列，排序，重复的仅取一个
$ last | cut -d ' ' -f1 | sort | uniq

//承上例，计数
$ last | cut -d ' ' -f1 | sort | uniq -c
1
2
3
4
5

# wc

计数，统计文件有多少字、行、字符

$ wx [-lwm]
1

其中：

-w：仅列出字

-l：仅列出行

-m：仅列出字符

使用示例：

$ cat /etc/passwd | wc
24      36    1099
分别代表：行，字，字符数
1
2
3

# 双向重定向

# tee

数据流输出到标准输出的同时，也可以保存到文件中做后续分析

$ tee [-a] file
1

其中：

-a：内容在file后面累加

使用示例：

//last输出到last.list文件中，同时，在屏幕显示第一列
$ last | tee last.list | cut -d ' ' -f1
1
2

# 字符转换命令

# tr

删除或者替换一段信息中的文字

$ tr [-ds] SET1 ...
1

其中：

-d：删除信息当中的SET1字符串

-s：替换重复的字符

使用示例：

//将/etc/passwd中所有的:删除
$ cat /etc/passwd | tr -d ':'

//将/etc/passwd中所有的小写字母大写
$ cat /etc/passwd | tr -s [a-z] [A-Z]
1
2
3
4
5

# col

$ col [-xb]
1

其中：

-x：将tab键转换为对等的空格

-b：文字内有反斜杠，保留反斜杠最后接的字符

# 切割命令

# split

$ split  [-bl] file PREFIX
1

其中：

-b：切割成的文件的大小，可加单位：b、k、m等

-l：以行进行切割

PREFIX：生成的切割文件的前缀

使用示例：

$ split -b 300k /etc/passwd passwd

//合并
$ cat passwd* >> passwdback
1
2
3
4

# 参数代换

# xargs

xargs接收管道符之前的输出，然后，将其默认按空格分隔，分隔后的数据作为xargs后命令的参数。

xargs [-epnd] command
1

其中：

-e：EOF，结束字符串

-p：执行命令时，询问用户

-n：每次传入几个参数

-d：分隔符，将接收的字符串按指定分隔符分开
